# SlipStream MD

Technical specification for SlipStream Analytics, a developer-productivity analytics platform that connects AI usage, GitHub activity, Jira work, and identity resolution into a single operating view.

## What Is Included

- Product overview and core value proposition.
- Backend and frontend architecture.
- Database schema and identity model.
- API, sync, configuration, and deployment details.
- Implementation notes intended to support recreation or review of the SlipStream system.

## Primary Document

Start with [SPECIFICATION.md](SPECIFICATION.md). It is the canonical technical reference for this repository.

## Status

This repository is documentation-only. It does not contain the SlipStream application source code.
